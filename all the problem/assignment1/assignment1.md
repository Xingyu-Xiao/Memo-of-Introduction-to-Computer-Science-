### 1A. Theatre Square

math, 1000, https://codeforces.com/problemset/problem/1/A



思路：



##### 代码

```python
x, y, z = input().split()
x, y, z = int(x), int(y), int(z)
if x % z == 0:
    a = x//z
    if y % z == 0:
        b = y//z
        print(a * b)
    else:
        b = y // z + 1
        print(a * b)
if x % z != 0:
    a = x // z+1
    if y % z != 0:
        b = y//z+1
        print(a * b)
    else:
        b = y // z
        print(a * b)


```

### 112A. Petya and Strings

implementation, strings, 1000, http://codeforces.com/problemset/problem/112/A



思路：



##### 代码

```python
a = input()
b = input()
a = a.lower()
b = b.lower()
la = list(a)
lb = list(b)
if la == lb:
    print(0)
else:
    for i in range(len(la)):
        if la[0] == lb[0]:
            del lb[0]
            del la[0]
    lab = [la[0], lb[0]]
    lab_sort = sorted(lab)
    if lab == lab_sort:
        print(-1)
    else:
        print(1)


```
